# 全屏圆形进度组件


### props

|  属性 | 说明 | 类型 | 默认值 |
| ------------ | ------------ | ------------ | ------------ |------------ |
| pageTitle | 顶部标题 | String  | 无 , 可不传 |
| isShow | 显示隐藏 ,  使用v-model双向绑定 | Boolean | false
| percent | 进度值 | Number  |

### 事件名
|  事件名 | 说明 | 返回值 |
| ------------ | ------------ | ------------ |
| successFn | 进度百分之百后的回调 | 无 |
